FACTORY RESET GUIDE
Step-by-step walkthrough

Wipe your Mac.
Clean slate.

A complete guide to factory resetting your Mac — erasing all data, removing your Apple ID, and restoring macOS from scratch.

⚠️
Back up before you begin This process permanently deletes all files, apps, and personal data from your Mac. Make sure you have a complete backup via Time Machine or iCloud before proceeding.
Apple Silicon (M1/M2/M3/M4)
Intel Mac
Apple Silicon — Complete Reset Procedure
01
Back Up Your Data
CRITICAL
Connect an external drive and open Time Machine (System Settings → General → Time Machine) to create a full backup. Alternatively, ensure iCloud Drive sync is complete. All data will be permanently erased — there is no undo.
02
Sign Out of Apple ID & iCloud
CRITICAL
Go to System Settings → [Your Name], scroll to the bottom and click Sign Out. Choose whether to keep a copy of iCloud data on this Mac. This removes Activation Lock so the next owner can set up the Mac. Also sign out of iMessage (Messages → Settings → iMessage → Sign Out).
03
Unpair Bluetooth Devices
OPTIONAL
Go to System Settings → Bluetooth. Click the ⓘ next to each device and select Forget This Device. This is especially important if you're passing the Mac to someone else.
04
Use Erase All Content and Settings
MACOS 12+
On macOS Monterey or later, go to System Settings → General → Transfer or Reset → Erase All Content and Settings. Enter your admin password and Apple ID password when prompted. The Mac will erase itself and reboot to the Setup Assistant automatically.

This is the easiest method for Apple Silicon Macs.
05
Alternative: Boot into Recovery Mode
IF NEEDED
If you can't access System Settings, shut down the Mac completely. Then press and hold the Power button until you see "Loading startup options." Click Options → Continue to enter macOS Recovery. From the Recovery menu, choose Disk Utility to erase the drive, then use Reinstall macOS.
Power Button (hold) → Options → Continue → Disk Utility / Reinstall macOS
06
Erase the Disk (in Disk Utility)
IN RECOVERY
In Disk Utility, select Macintosh HD in the sidebar (View → Show All Devices). Click Erase, set the format to APFS, and confirm. After erasing, also delete the Macintosh HD - Data volume if present. Then quit Disk Utility to return to the Recovery menu.
07
Reinstall macOS
SYSTEM
From the Recovery menu, select Reinstall macOS (the version shown will be the latest compatible with your Mac). Connect to Wi-Fi when prompted. The download and installation will take 30–60 minutes. The Mac will restart several times — do not interrupt power during this process.
08
Complete Setup or Hand Off
FINAL STEP
After reinstallation, the Mac boots to the Hello / Setup Assistant screen — exactly as it was when new. If keeping the Mac yourself, follow the prompts to sign in with your Apple ID and restore from your backup. If giving the Mac away, leave this screen for the new owner to complete setup with their own Apple ID.
Intel Mac — Complete Reset Procedure
01
Back Up Your Data
CRITICAL
Use Time Machine or iCloud to back up all files before proceeding. All data will be permanently destroyed.
02
Sign Out of Apple ID & iMessage
CRITICAL
Go to System Preferences → Apple ID → Overview → Sign Out. Also open Messages → Preferences → iMessage → Sign Out.
03
Boot into macOS Recovery
KEY COMBO
Restart your Mac. Immediately hold ⌘ Command + R until you see the Apple logo or a spinning globe. This boots into macOS Recovery. On some Macs, holding Option/Alt shows a boot picker.
Restart → Hold ⌘ + R immediately
04
Open Disk Utility & Erase
ERASE DISK
In Recovery, open Disk Utility. Click View → Show All Devices. Select Macintosh HD and click Erase. Format: APFS (or Mac OS Extended Journaled for older macOS). Click Erase Volume Group or Erase.
05
Reinstall macOS
SYSTEM
Quit Disk Utility. Back in Recovery, select Reinstall macOS. Connect to Wi-Fi. The installer will download macOS and install it. The Mac will restart multiple times — keep it plugged in and don't interrupt.
06
Complete Setup or Hand Off
FINAL STEP
Once complete, the Mac shows the Hello setup screen. Sign in or leave for the next user. Your Mac is now fully wiped and reset to factory condition.
